<h1>Mini putty</h1>
<p>Es una mini aplicacion que corre en nodejs y permite ejecutar comandos en el servidor, desde el navegador</p>
<a href="http://miniputty.heroku.com">Miniputty =)</a>
<br>

<pre>cat ../proc/meminfo</pre>
